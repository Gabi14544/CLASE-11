# Ejercicio 2
Desarrollar un paquete llamado banco que contenga un archivo de inicialización y dos módulos: cuentas.py y gestion.py. 

En el primer módulo, estructurar la clase CuentaBancaria con los atributos titular y saldo, además crear 2 métodos para realizar depósitos y retiros mediante validaciones lógicas básicas. 

En el segundo módulo, definir la función crear_cuenta encargada de solicitar los datos iniciales al usuario mediante input, garantizar que el saldo inicial sea un valor numérico entero no negativo antes de instanciar y retornar el objeto. 

Finalmente, crear un archivo main.py externo que importe esta función, inicialice el registro de la cuenta y proporcione un menú interactivo en un ciclo continuo que valide las entradas de depósitos y retiros.
